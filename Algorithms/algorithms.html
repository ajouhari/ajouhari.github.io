    General Algorithms

    Many of our everyday tasks are rather menial, and can easily be automated or assisted by computer algorithms. Available is a list of algorithms that serve many different purposes. Each are labelled by their complexity, instructions, and the date at which they were created.

    Search Algorithms

    Search algorithms have two primary function: to find if a path exists between two locations, and if so, to find the shortest path between them. Here I will code and review multiple search algorithms, from a brute-force method, to a well-designed a-star search algorithm.

    Sort Algorithms
